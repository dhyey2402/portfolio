# Personal Portfolio Website

A modern, stylish, and fully responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🎨 **Modern Design** - Clean, minimalist design with smooth animations
- 📱 **Fully Responsive** - Works perfectly on all devices (desktop, tablet, mobile)
- ⚡ **Fast & Lightweight** - Optimized for performance
- 🎯 **Smooth Scrolling** - Enhanced navigation experience
- 🌈 **Beautiful Animations** - Subtle animations and transitions
- 📧 **Contact Form** - Ready-to-use contact form
- 🎭 **Interactive Elements** - Hover effects and interactive components

## Sections

1. **Hero Section** - Eye-catching introduction with call-to-action buttons
2. **About** - Personal information and statistics
3. **Skills** - Showcase of technical skills and technologies
4. **Projects** - Portfolio of featured projects
5. **Contact** - Contact information and form

## Getting Started

### Prerequisites

No build tools or dependencies required! Just a modern web browser.

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! Your portfolio is ready.

### Customization

#### Personal Information

1. **Name & Title**: Update in the hero section (`index.html`):
   ```html
   <h1 class="hero-name">Your Name</h1>
   <h2 class="hero-title">Full Stack Developer</h2>
   ```

2. **About Section**: Edit the about text in `index.html`:
   ```html
   <p>Your personal description here...</p>
   ```

3. **Contact Information**: Update contact details in the contact section:
   ```html
   <a href="mailto:your.email@example.com">your.email@example.com</a>
   <a href="tel:+1234567890">+1 (234) 567-890</a>
   ```

4. **Social Links**: Add your social media profiles:
   ```html
   <a href="your-linkedin-url" target="_blank">...</a>
   ```

#### Projects

Replace the example projects with your own:

1. Update project titles and descriptions
2. Add your project links (live demo and GitHub)
3. Update project tags/technologies used
4. Optionally add project images

#### Skills

Modify the skills section to match your expertise:

1. Add or remove skill categories
2. Update skill items with your technologies
3. Icons are from Font Awesome (already included)

#### Colors

Customize the color scheme in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... other colors */
}
```
